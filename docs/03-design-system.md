
# AME Bazaar Design System

## Purpose

This document defines the visual language and user interface standards for the AME Bazaar Digital Platform.

Its purpose is to ensure consistency across every page, component, feature, and future application.

Every designer, developer, AI assistant, and content creator must follow this document.

---

# Design Philosophy

The AME Bazaar design language is inspired by premium global retail brands.

Inspirations

* Apple
* Zara
* Nike
* Uniqlo
* Westside

The interface should always feel:

* Premium
* Minimal
* Elegant
* Modern
* Spacious
* Professional
* Trustworthy

Avoid:

* Template-like layouts
* Heavy gradients
* Excessive animations
* Visual clutter
* Overuse of shadows
* Loud colors
* Flashy banners

---

# Design Principles

Every interface should prioritize:

* Simplicity
* Readability
* Accessibility
* Performance
* Mobile-first design
* Clear visual hierarchy
* Conversion optimization
* Fast loading

---

Navy Blue

Used for:

* Logo
* Header
* Primary Buttons
* Links
* Important Sections

---

## Accent Color

Golden

Used for:

* Highlights
* Icons
* Badges
* Hover States
* Call To Actions

---

## Neutral Colors

White

Primary background.

Light Gray

Section backgrounds.

Dark Gray

Secondary text.

Black

Headings.

---

# Typography

Typography should be clean, modern and highly readable.

Heading Style

* Large
* Bold
* High contrast

Body Text

* Comfortable reading size
* Generous line height
* High readability

Text Alignment

* Left aligned
* Avoid justified text

---

# Spacing System

Use generous white space.

Maintain consistent spacing throughout the website.

Visual rhythm is more important than fitting more content.

---

# Grid System

Desktop

Maximum content width:
1200px–1280px

Tablet

Responsive layout with reduced spacing.

Mobile

Single-column layout.

All sections should stack naturally.

---

# Buttons

Primary Button

* Navy background
* White text
* Rounded corners
* Clear hover state

Secondary Button

* White background
* Navy border
* Navy text

Text Button

Minimal underline or simple hover effect.

---

# Cards

Cards should be:

* Clean
* Minimal
* Soft border
* Light shadow (if needed)
* Spacious padding

Never overcrowd cards.

---

# Images

Use:

* High-resolution images
* Natural lighting
* Clean backgrounds
* Real products
* Authentic store photography

Avoid:

* Stock-looking images
* Low-quality photos
* Pixelated images
* Busy backgrounds

---

# Icons

Use simple outline icons.

Icons should support content rather than dominate it.

Maintain consistent icon style throughout the website.

---

# Forms

Every form should include:

* Clear labels
* Helpful placeholders
* Accessible focus states
* Inline validation
* Large tap targets on mobile

Keep forms as short as possible.

---

# Navigation

Navigation should be:

* Simple
* Predictable
* Easy to scan
* Mobile-friendly

Users should reach any important page within three clicks.

---

# Components

The following components must remain visually consistent across the platform:

* Header
* Navigation

---

# Accessibility

The design must comply with modern accessibility standards.

Requirements

* Proper color contrast
* Keyboard navigation
* Visible focus indicators
* Screen reader compatibility
* Semantic HTML
* Alt text for images
* Responsive typography

Accessibility should never be sacrificed for aesthetics.

---

# Performance

The design system should prioritize performance.

Avoid:

* Large animations
* Heavy JavaScript
* Unnecessary libraries
* Oversized images

Target excellent Core Web Vitals.

---

# Responsive Design

Every component must work consistently across:

* Mobile
* Tablet
* Laptop
* Desktop
* Large Displays

Design mobile-first, then progressively enhance for larger screens.

---

# Consistency Rules

Every page, component, template, WooCommerce screen, blog post, landing page, and future feature must follow this design system.

New components should extend the existing system instead of creating new visual patterns.

This document is the permanent visual foundation of the AME Bazaar Digital Platform.
